# 📱 如何在iPhone上下载和使用Bark软件

1. **通过iPhone上的App Store下载软件**  
   - 软件名称为 **Bark**。

2. **下载完成后**  
   - 对软件进行注册，并复制你的 **Bark专属URL**。

3. **功能简介**  
   - **Bark** 是用来发送通知的。
